## Programs Included

### AgeCalculator.java
- Calculates age based on the given date of birth
- Uses conditional logic and input handling

### Date-From-Day-Number.java
- Converts a day number into the corresponding date
- Demonstrates control flow and logical branching

### MessageDecoder.java
- Decodes an encrypted message using character shifting
- Shows string manipulation and loop logic

### MirrorImageMatrix.java
- Generates the mirror image of a matrix
- Demonstrates 2D arrays and nested loops

### Sort-Words-Alphabetically.java
- Sorts words in alphabetical order
- Uses string comparison and array processing
